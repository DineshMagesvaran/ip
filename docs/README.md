# User Guide

## Features 

### List
Feature that enables user to add items to a list and retrieve the list

## Usage

### 'list' - displays all items currently in the list
			 Example: list
	Expected outcome: 1. apples
					  2. oranges
					 
### <any item> - adds the item to the back of the list
			 Example: bananas
	Expected outcome: 1. apples
					  2. oranges
					  3. bananas
